# Audio files

Drop the following 5 files into this folder. Missing files render silent —
no crash — so you can start with just the music bed and add SFX later.

| File             | Used for                         | Length      |
|------------------|----------------------------------|-------------|
| `music-bed.mp3`  | Whole video (looped/trimmed)     | 30s+        |
| `whoosh.mp3`     | Scene transitions                | ~0.5–1s     |
| `keytype.mp3`    | Identity commitment typewriter   | ~3–4s       |
| `chime.mp3`      | Register tx confirmation         | ~1s         |
| `success.mp3`    | Vote submitted                   | ~1–1.5s     |

## Where to get them (free / royalty-free)

- **Pixabay Music** (https://pixabay.com/music/) — free, no attribution.
  Search "minimal tech" or "ambient electronic" for the music bed.
- **Freesound** (https://freesound.org/) — CC0 / CC-BY. Search "ui whoosh",
  "mechanical keyboard", "success chime", "notification".
- **Zapsplat** (https://zapsplat.com/) — free with account.
- **Uppbeat** (https://uppbeat.io/) — free tier for creators, great for
  YouTube monetization-safe music.

## Suggested vibe

Dark, minimal, slightly synthy — matches the app's indigo-on-black
aesthetic. Think "privacy tech product demo", not "hype EDM drop".
