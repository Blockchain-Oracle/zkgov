# Audio Files

Place the following audio files here, then set `AUDIO_ENABLED = true` in `src/compositions/ZkGovQuickstart/index.tsx`.

| File | Use | Length | Notes |
|------|-----|--------|-------|
| `music-bed.mp3` | Background music, full 30s | 30s+ | Looped at vol 0.25. Ambient/lo-fi works best. |
| `whoosh.mp3` | Scene transition SFX (plays 6×) | 0.5–1s | UI whoosh / swipe sound |
| `keytype.mp3` | Identity commitment typewriter | 3–4s | Keyboard typing sound |
| `chime.mp3` | Register tx confirmed | ~1s | Soft confirmation chime |
| `success.mp3` | Vote submitted | ~1.5s | Positive success sound |

## Free sources

- https://freesound.org — search "whoosh", "chime", "success"
- https://pixabay.com/music — royalty-free music beds
- https://uppbeat.io — free with attribution
